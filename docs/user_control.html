Управление пользователями

Интерфейс управления пользователями позволяет добавлять/удалять/редактировать пользователей системы.

При первоначальной загрузке отображается в виде таблицы список уже внесенных в систему пользователей и информация о штрихкоде, ФИО, логине, пароле, роли каждого пользователя. Последняя колонка таблицы содержит функциональные кнопки для работы с записями таблицы.

По умолчанию пароли пользователей скрыты под символами звездочек. При наведении курсора на область пароля он принимает форму указательного пальца. В этот момент нажатие кнопки мыши отображает пароль под курсором.

Заголовок столбца ролей содержит выпадающий список имеющихся в системе ролей. Выбор той или иной позиции в этом списке включает фильтр и отображает в таблице только пользователей с выбранной ролью. Для сброса фильтра необходимо выбрать в списке позиции без определенной роли, содержащую символы "------".

- кнопка добавления пользователя. При нажатии на нее в конце таблицы появляется строка с редактируемыми полями. После внесения всех необходимых данных необходимо нажать кнопку в последней ячейке строки. Если в процессе добавления пользователя администратор передумал, то можно отменить действие добавления, нажав на кнопку .

Для редактирования данных того или иного пользователя необходимо нажать кнопку в последней ячейке соответствующей строки таблицы. После этого все информационные ячейки строки станут редактируемыми, а в последней ячейке появятся следующие кнопки:

- сохранить внесенные изменения;

- отменить внесенные изменения;

- удалить пользователя.

ВНИМАНИЕ! Удаление пользователя не влечет за собой полное удаление информации о нем в базе данных, а лишь помечает соответствующую запись, как удаленную. В связи с этим для сохранения уникальности логинов пользователей добавление нового пользователя с существующим в системе логином, даже если пользователь был удален ранее, запрещено. То же правило действует и для штрихкодов.
