Руководство пользователя "Администратор"

Система "Проволока" предназначена для учета движения металлов (алюминий и сталь) в процессе изготовления элементов грозотроса.

Система поддерживает обработку состояний следующих видов материалов:

  • Алюминиевая катанка
  • Алюминиевая проволока
  • Стальная проволока
  • Плакированная проволока

Изменение состояний указанных изделий в результате выполнения операций системы приведено на схемах ниже.

В системе имеются следующие роли:

  • Администратор
  • Кладовщица
  • Мастер цеха
  • Начальник цеха
  • Оператор ОТК
  • Снабженец
  • Руководитель

Административные функции:

  1. Управление пользователями
  2. Распечатать штрихкод

Функции общего назначения:

  1. Сканер

Функции АРМ кладовщицы:

  1. Внести данные
  2. Отгрузить на производство

Функции АРМ мастера цеха:

  1. Взвесить
  2. Установить на линию плакирования
  3. Отгрузить на склад

Функции АРМ оператора ОТК:

  1. Проверить изделие после плакировки

Функции АРМ снабженца:

  1. Отгрузить отходы

Отчеты, доступные руководству предприятия:

  1. Сводный отчет по линии плакирования
  2. Сменный отчет по линии плакирования
  3. Месячный отчет
